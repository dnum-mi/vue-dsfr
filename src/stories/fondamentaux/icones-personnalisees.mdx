## Icônes Remix icons

Il est possible d’utiliser facilement toutes les icônes [Remix Icon](https://remixicon.com/) grâce à
[@iconify/vue](https://iconify.design/docs/icon-components/vue/) : le composant [`VIcon`](https://vue-ds.fr/composants/VIcon) est enregistré globalement si la bibliothèque est
utilisée en tant que plugin.

### Ajouter des icônes personnalisées

Pour ajouter des icônes personnalisées, se reporter à [icon-sets.iconify.design](https://icon-sets.iconify.design/) pour connaître les noms des icônes.
